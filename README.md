# Electron TodoList - 分组管理版

一个功能强大的 TodoList 桌面应用，支持分组管理和多窗口操作，灵感来自微软 Sticky Notes。

![Version](https://img.shields.io/badge/version-2.3-blue)
![Electron](https://img.shields.io/badge/Electron-27.0.0-green)
![License](https://img.shields.io/badge/license-MIT-orange)

## ✨ 功能亮点

### 🎯 核心功能
- **📋 分组管理** - 创建多个待办分组，分类管理任务
- **🪟 多窗口支持** - 每个分组独立窗口，可同时打开多个
- **📌 窗口置顶** - 分组窗口可以固定在最前面，方便随时查看
- **🎯 智能标题栏** - 分组窗口标题栏自动隐藏，鼠标靠近时显示
- **📊 进度可视化** - 实时显示每个分组的完成进度
- **🔔 系统托盘** - 关闭窗口最小化到托盘，双击托盘图标恢复
- **💾 自动保存** - 所有操作自动保存，数据永不丢失
- **✏️ 快速编辑** - 双击即可编辑分组或待办名称
- **🗑️ 管理模式** - 一键进入管理模式，快速删除分组

### 🎨 界面特色
- **简约现代** - 清新的设计风格，赏心悦目
- **渐变标题栏** - 紫色渐变，优雅大方
- **流畅动画** - 平滑的过渡效果，提升体验
- **无边框窗口** - 自定义标题栏，专业美观
- **响应式设计** - 支持调整窗口大小

## 🚀 快速开始

```bash

```

## 📖 使用指南

### 创建分组
1. 在输入框中输入分组名称（如："工作"、"学习"、"生活"）
2. 按 **Enter** 键或点击 **+** 按钮
3. 分组立即出现在列表中

### 打开分组
- **点击分组** - 在独立窗口中打开
- **可同时打开多个分组**
- **每个窗口独立操作**

### 管理待办
- **添加** - 输入内容后按 Enter
- **完成** - 点击复选框
- **编辑** - 双击文本内容
- **删除** - 悬停后点击 × 按钮

### 编辑分组
- **双击分组名称** - 进入编辑模式
- **Enter** 保存，**Esc** 取消

### 删除分组
1. 点击 **"管理模式"** 按钮
2. 点击要删除的分组
3. 确认删除操作
4. 点击 **"完成"** 退出管理模式

## 💾 数据存储

### 存储位置
- **Windows**: `%APPDATA%\electron-todolist\data\groups.json`
- **macOS**: `~/Library/Application Support/electron-todolist/data/groups.json`
- **Linux**: `~/.config/electron-todolist/data/groups.json`

### 数据格式
```json
{
  "groups": [
    {
      "id": "唯一ID",
      "name": "分组名称",
      "todos": [
        {
          "id": "待办ID",
          "text": "待办内容",
          "completed": false,
          "createdAt": 1699999999999,
          "updatedAt": 1699999999999
        }
      ],
      "createdAt": 1699999999999,
      "updatedAt": 1699999999999
    }
  ]
}
```

### 数据备份
直接复制 `groups.json` 文件即可备份所有数据。

## ❓ 常见问题

### Q: 分组窗口关闭后数据会丢失吗？
**A**: 不会。所有数据自动保存，重新打开即可看到。

### Q: 可以创建多少个分组？
**A**: 没有限制，但建议保持在 10 个以内便于管理。

### Q: 可以同时打开多个分组吗？
**A**: 可以！每个分组都可以在独立窗口中打开。

### Q: 删除分组可以恢复吗？
**A**: 删除后无法恢复，删除前会有确认提示。

### Q: 如何备份数据？
**A**: 直接复制 `groups.json` 文件即可。

### Q: 如何完全退出程序？
**A**: 右键系统托盘图标，选择"退出"。
