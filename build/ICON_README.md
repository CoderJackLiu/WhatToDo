# 图标说明

## 图标文件

本目录包含 TodoList 应用的所有图标文件：

- **icon.ico** - Windows 应用主图标（包含多个尺寸：16x16, 32x32, 48x48, 64x64, 128x128, 256x256）
  - 用于：安装包图标、快捷方式图标、控制面板卸载程序图标、窗口图标
- **icon-16.png** ~ **icon-256.png** - 各尺寸 PNG 图标（用于其他用途）

## 图标设计

图标采用紫色渐变主题（#667eea → #764ba2），与应用界面风格保持一致：
- 背景：紫色渐变圆角矩形
- 图标：白色纸张 + 待办清单线条 + 复选框

## 使用位置

1. **系统托盘图标** - 最小化到托盘时显示（16x16）
2. **窗口图标** - 应用窗口标题栏和任务栏显示
3. **快捷方式图标** - 桌面和开始菜单快捷方式
4. **安装包图标** - NSIS 安装程序显示
5. **控制面板图标** - 卸载程序时显示

## 重新生成图标

如果需要修改图标设计，编辑 `build/generate-icon.js` 中的 SVG 代码，然后运行：

```bash
node build/generate-icon.js
```

## 依赖

- `sharp` - 图像处理库
- `to-ico` - PNG 转 ICO 格式
